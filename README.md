1️⃣ What is the difference between var, let, and const?

Ans: var: var can be redeclared and reassigned. It also shows undefined when hoisted. It is not block scoped.

let: It can be reassigned but not redeclared. It is block scoped.

const:It can not be redeclared and reassigned. It is also block scoped.
